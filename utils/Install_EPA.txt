Pour que l'application de gestion du recrutement en CPES/CPGE tourne, il faut :
1/
	Installer Python 3.6 (ou supérieur), avec option PIP et ajout dans PATH : se fait à partir du site "www.python.org" 
	menu "downloads". 
2/
	Lancer les commandes suivantes (en ligne de commande) pour installer les bibliothèques :
		python -m pip install --upgrade cherrypy
		python -m pip install --upgrade parse
		python -m pip install --upgrade lxml
		python -m pip install --upgrade PyPDF2

Si tout s'est bien passé, le reste fonctionnera à merveille...
